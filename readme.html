使い方

屋根材の左右均等割付、必要枚数、キャップ数、けがき寸法を確認するための説明ページです。

計算へ戻る

概要

働き幅 333mm の屋根材を左右均等に割付し、必要枚数、キャップ数、けがき寸法を計算します。

使い方

  1. 屋根の横幅 W を入力します。
  2. 働き幅、最小端幅、最大端幅 を必要に応じて変更します。ハイロックでは 理想の端幅 も指定できます。
  3. 計算結果は自動で更新されます。
  4. コピー ボタンで結果をクリップボードへ保存できます。
  5. 印刷 ボタンで A4 印刷プレビューを表示できます。

見方

  • おすすめ端幅 は左右に入る同じ寸法です。
  • 中央の定尺枚数 は働き幅のまま使う枚数です。
  • 必要枚数 は中央材と端部用を含めた枚数です。
  • キャップ数 は材どうしの境界数です。
  • けがき位置 は割付位置から右へ 25mm ずらした位置です。

注意点

  • 端幅が 140mm 以下の場合は、端部材を 1 枚で取り合わせます。
  • 端幅が 140mm を超える場合は、端部用として 2 枚追加します。
  • サービスワーカーは file:// では登録されません。HTTPS または localhost 上で有効になります。
  • 入力値に不正がある場合は、計算結果に入力確認が表示されます。
