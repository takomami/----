# 屋根割付

このプロジェクトは、働き幅 333mm の屋根材を左右均等に割付し、必要枚数、キャップ数、けがき寸法を計算するシンプルなウェブアプリです。

## 概要

- `roof-layout.html`：計算アプリ本体
- `manifest.webmanifest`：PWA 用マニフェスト
- `sw.js`：オフラインキャッシュ用サービスワーカー
- `roof-icon.svg`：アプリアイコン

## 特長

- 入力値に応じた割付結果を即時表示
- おすすめプランと第2プランを比較
- けがき寸法一覧を自動生成
- 結果をクリップボードにコピー可能
- A4 印刷レイアウト対応
- PWA としてキャッシュ可能（HTTPS または localhost 環境で有効）

## 使い方

1. `roof-layout.html` をブラウザで開きます。
2. 以下の値を入力します。
   - 屋根の横幅 W
   - 働き幅
   - 理想の端幅
   - 最小端幅
   - 最大端幅
3. 計算結果が自動的に更新されます。
4. `コピー` ボタンで結果をクリップボードに保存できます。
5. `印刷` ボタンで A4 印刷プレビューを表示できます。

## 注意点

- サービスワーカーは `file://` では登録されません。HTTPS または localhost 上で実行してください。
- 入力値に不正がある場合はエラー表示になり、計算は行われません。

## ファイル説明

- `roof-layout.html`
  - 入力 UI、結果表示、計算ロジックを含む単一ページアプリ
- `manifest.webmanifest`
  - PWA の名称、アイコン、テーマカラーなどを定義
- `sw.js`
  - キャッシュ戦略を実装し、アプリのオフライン利用を補助

## 追加情報

- `idealEdge` は初期値 150mm です。
- 端幅が 140mm 以下の場合は端部材を 1 枚で取り合わせ、140mm 超の場合は 2 枚追加します。
- 表示されるキャップ数は材どうしの境界数を表します。
